# Teslo Shop

1. Clonar el repositorio
2. Levanatar base de datos con docker compose up -d
3. correr back-end instalar instacias con npm i y verificar que este en el puerto 3000 con npm run:start dev
4. correr front-end npm run dev 
5.  crear archivo .env y agregar .en a git ignore
6. hacer copia de .env con .env.template
7. crear portafolio para llamar a API
